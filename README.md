# linux_setup_script
Script to run after fresh linux install

Select if you are workign with a VM or Real system.

This scrtip sets the timezone to Est, install dotfile, and XCP-NG tools(if on vm) and fully update / upgrade your system.

### Clone the repo and run setup.sh
```
git clone "https://github.com/acebmxer/linux_setup_script.git"
cd linux_setup_script && chmod +x * && bash setup.sh
```
